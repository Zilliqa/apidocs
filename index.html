    Introduction

    JSON-RPC is a remote procedure call protocol encoded in JSON. You can use this API to access data from the Zilliqa nodes. The JSON-RPC API server runs on:

    Chain(s) URL(s)
    Zilliqa Mainnet https://api.zilliqa.com/
    Developer testnet https://dev-api.zilliqa.com/
    Local testnet http://localhost:4201/

    All API calls are POST requests.

    All requests follow the standard JSON-RPC format and include 4 variables in the data object:

    Data object Example
    id e.g. "1"
    jsonrpc e.g. "2.0"
    method e.g. "GetBalance"
    params e.g. ["1"]

    Blockchain-related methods

    GetNetworkId

    Example request:

    curl -d '{
    "id": "1",
    "jsonrpc": "2.0",
    "method": "GetNetworkId",
    "params": [""]
}' -H "Content-Type: application/json" -X POST "https://api.zilliqa.com/"

    const NetworkId = await zilliqa.network.GetNetworkId();
console.log(NetworkId);
